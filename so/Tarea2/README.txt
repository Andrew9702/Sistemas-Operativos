DESCRIPCION:

Debido a problemas que tuve para leer los argumentos de las funciones, cree alternativas para 1 o 2 argumentos.

En LS se debe escribir: "ls" sin argumentos, "ls1 <argumento>" para ejecutarlo con un argumento, "ls2 <argumento1> <argumento2>" para
ejecutarlo con 2 argumentos.

En CAT se debe escribir "cat <archivo>" para un argumento y "cat1 <archivo1> <archivo2>" para dos argumentos.

Los demas comandos (pwd, cp, grep) funcionan con regularidad.

Para salir de la aplicacion de shell se debe escribir "exit".
